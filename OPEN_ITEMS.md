## ✅ APPROVED → A/B TEST RUNNING — `/platform-research` (Jun 20, 2026): LinkedIn first-comment link penalty
_**DECISION (Jun 20, 2026): Ginny approved testing, not a rule change.** Running Option 1 (timed-delay first comment) as a ~3-week A/B, logged in `linkedin_performance_tracker.md` → "Active Experiment — First-comment link timing." No CLAUDE.md / week_build_spec / deep-dive edit until the data settles it (~12 posts). Revisit at the next `/platform-research` run or when the log fills. Original proposal below for reference._

- **Platform:** LinkedIn (feed).
- **Finding:** Multiple 2026 sources now report LinkedIn has expanded its external-link penalty beyond the post body to **comments containing external links (visibility reportedly down up to ~80%)** and that the algorithm detects **"bridge behavior"** (a post visibly engineered to funnel readers to a link sitting in the first comment) and applies a reach penalty to the post itself. _(DIRECTIONAL — the specific 80% figure is single-vendor; the direction is corroborated across [gromming](https://gromming.com/blog/linkedin-external-links-penalty), [melaniegoodman](https://melaniegoodmanlinkedinconsultant.substack.com/p/linkedin-algorithm-2026-reach-topic-authority), [voketa](https://www.voketa.com/blog/how-to-avoid-linkedins-hidden-penalties), [dataslayer.ai Feb 2026](https://www.dataslayer.ai/blog/linkedin-algorithm-february-2026-whats-working-now). No first-party LinkedIn confirmation.)_
- **ToS + brand-fit verdict:** No ToS risk (this is about *avoiding* a penalty, not gaming one); fully brand-safe.
- **Measured-data context (the override):** PBS already reframed the first-comment Substack link as a **conversion/tracking mechanism, NOT a reach multiplier** (CLAUDE.md, Jun 18 2026; W18 logged 4-5 Substack subs via it). So PBS is not losing *reach it was counting on*. The genuinely-new question is whether the comment-link now **actively suppresses the host post** (the "bridge behavior" claim). PBS has no logged A/B on this; measured data is silent, so this is a real open call rather than a conflict the data already settles.
- **Proposed rule change (pick one; all preserve the conversion path while removing any suppression risk):**
  1. **Keep the first-comment link, add a delay** — post the link as a comment a few minutes *after* the post clears its first-engagement window, so the post is ranked before any link is attached. Lowest-disruption; testable.
  2. **Route the cross-promo through the LinkedIn Newsletter / native Article surface** (which carries **no** link penalty) instead of a feed-comment link, on weeks where the deep dive maps to the newsletter.
  3. **Standalone-insight repost** — on the highest-reach Thursday post, drop the comment link entirely and instead lift the Substack core point into the post body as native value (the "turn the link into a post" 2026 best practice), keeping the link only on lower-stakes posts.

- **Recommendation:** Lowest-regret path is **Option 1 (timed-delay comment) as a 2-3 week A/B**, logged via `/log-metrics`, before any CLAUDE.md edit — let PBS's own data settle it, since the finding is directional and the measured-data override is the house rule.
