# Competitor / Like-Brand Ad Teardown

_Created Jun 23, 2026. The running log of what direct competitors and benefits-advisor brands run as **paid ads**, built from the public **LinkedIn Ad Library** (and Meta Ad Library when that platform is added). Maintained monthly by `/ads-research`. Feeds the concepts in `paid_ads_bank.md`. Benchmark set (Ginny's pick): **independent PBM auditors / pharmacy-benefits consultants** and **benefits brokers / advisor thought leaders**._

**How to read it:** the LinkedIn Ad Library (linkedin.com/ad-library) and Meta Ad Library (facebook.com/ads/library) show every active ad a page runs, with start dates. **Ad longevity is the single best public proxy for "what's working"** — a brand does not keep paying to run a losing ad. So log start date / how long it's run; a long-running ad is a tested winner worth learning from. Also note offer, hook, format, and CTA.

**No fabrication:** only log ads actually observed in the libraries, with the date checked. If a competitor runs no ads, log that too (a real signal: the space may be under-advertised, which is opportunity).
