# Platform Playbooks — best practices by channel (living doc)

_Created Jun 20, 2026. The current best-practice playbook for every channel PBS posts to (or may): **LinkedIn, Substack, X, YouTube Shorts, TikTok, Instagram, Facebook, Threads, YouTube (long-form), Reddit/Quora.** Refreshed **monthly** by `/platform-research` (`.github/workflows/monthly-platform-research.yml`), which web-researches each platform with adversarial verification, auto-applies format/spec/tactic updates here (dated in the Changelog), and proposes rule-level changes to `OPEN_ITEMS.md` for one-time approval._

## How this file is used (so findings actually get applied)
- **`/build-week` and `/clip-podcast` read this file as a required input.** They draft each platform's content (posts AND video clips) to the CURRENT best practices below. Update this file → the next build applies it automatically.
- **`week_build_spec.md` §8** gates conformance; `/critique` scores against it.
- **Two hard overrides:** (1) **PBS's measured data wins.** Where a generic best practice conflicts with PBS's logged WORKING/WEAK results (`linkedin_performance_tracker.md`), the measured data governs. (2) **Brand-fit filter.** PBS is a clinical-credibility B2B brand reaching CFOs / HR / brokers as Prescription Benefit Solutions. Reach that pulls the wrong audience or risks platform ToS is rejected, not chased. "Max reach" is in service of reaching plan sponsors, not vanity views.
- **Confidence tags:** CONFIRMED (first-party or consistent across many sources) vs. directional (vendor-aggregated, reverse-engineered). Treat percentages as directional.

## X (Twitter)
**Role:** amplification + policy/journalist relationships (Cuban/Ciaccia/Fein). NOT direct conversion. **Source of truth = `x_account_strategy.md` (PBS-measured).**
- Hooks: lead with the confrontation, no wind-up; the "named program · hidden extraction" recipe + "guess which…" close are PBS-proven (`x_recipe_post_bank.md`).
- Links in the FIRST REPLY, never the body. Body links now drop to **~0% engagement** for non-premium accounts since Mar 2025; text posts ~0.40%, video ~0.25%. Posting the link as a self-reply (second post in the thread) preserves the main post's reach. _(CONFIRMED — multiple 2026 sources, [Buffer](https://buffer.com/resources/twitter-timeline-algorithm/), [Sprout Social](https://sproutsocial.com/insights/twitter-algorithm/))_
- **Single long-form posts (expanded character limit) now distribute BETTER than multi-post threads** in the 2026 algorithm; dwell time on long-form >2 min is a strong positive signal (dwell weight +10, same as a bookmark). Reach for one substantive long post over a thread when the point is self-contained. _(CONFIRMED — [Buffer](https://buffer.com/resources/twitter-timeline-algorithm/), [teract.ai](https://www.teract.ai/resources/twitter-algorithm-2026))_ **PBS note:** still lead with the confrontation in the first line (the cold-feed gate); the long-form body carries the proprietary anchor + decode.
- 0-1 hashtags. Tagging > hashtagging.
- Visuals (Tue/Thu library assets, toolkit PNGs) outperform text-only. Embedded video that holds attention is amplified.
- Amplification is the lever: reaction speed when Cuban posts (weekend = his active window, PBS data). Measure reposts/quotes, not likes (likes-only = zero lift).

## LinkedIn (feed + newsletter)
**Role:** primary growth engine + the person-entity. **Source of truth = `linkedin_2026_algorithm_deep_dive.md` + CLAUDE.md (PBS-measured).**
- Ranking: dwell → comment *quality* → saves → reposts → reactions (weakest). First ~60 min is the gate. 8:30 AM ET (PBS A/B beat 10 AM).
- **2026 weighting (verified):** comments weigh ~15x a like; **multi-party comment threads (different users replying to each other) get ~5.2x more amplification** than posts without conversation depth. **A "Depth Score" (dwell + read-through) now governs ranking, not surface reactions.** _(CONFIRMED — [dataslayer.ai Feb 2026](https://www.dataslayer.ai/blog/linkedin-algorithm-february-2026-whats-working-now), [digitalapplied](https://www.digitalapplied.com/blog/linkedin-algorithm-2026-engagement-strategy-guide))_ **PBS lever:** write the post + first comment to invite a *reply-to-reply* (a closed question per the 6-beat CTE rule), not just a one-off comment.
- **Document/PDF (carousel) posts generate 2-3x more dwell** than single-image or short-text and are among the safest organic formats in 2026. _(CONFIRMED — [dataslayer.ai](https://www.dataslayer.ai/blog/linkedin-algorithm-february-2026-whats-working-now))_ Reinforces PBS's carousel + Library NN slots.
- **Native 9:16 video gets dual distribution (feed + recommendation video tab → non-followers).** Upload the file natively (never a YouTube/Vimeo link), hook in 3s, captions burned in, post from Ginny's personal profile. This is the bridge to the TikTok/Shorts clip pipeline.
- 3 hashtags (PascalCase), confrontational hooks (10-100x informational).
- **⚠️ EXTERNAL-LINK PENALTY EXPANDED (2026, conflicts with current CLAUDE.md first-comment rule — see OPEN_ITEMS proposal Jun 20, 2026):** body links cut reach ~18.8-60%; multiple 2026 sources now report LinkedIn **also suppresses comments containing external links (visibility down up to ~80%)** and detects "bridge behavior" (a post built to funnel to a link in the comment). _(DIRECTIONAL — single-vendor 80% figure; corroborated direction across [gromming](https://gromming.com/blog/linkedin-external-links-penalty), [melaniegoodman](https://melaniegoodmanlinkedinconsultant.substack.com/p/linkedin-algorithm-2026-reach-topic-authority), [voketa](https://www.voketa.com/blog/how-to-avoid-linkedins-hidden-penalties))_ **PBS measured-data override:** PBS's own first-comment cross-promo is logged as a CONVERSION/TRACKING lever (W18: 4-5 Substack subs), already reframed in CLAUDE.md as *not* a reach multiplier — so PBS does not lose reach it was counting on. The open question is whether the comment-link now actively *suppresses* the post; that is a rule-level call, proposed to OPEN_ITEMS (alternatives: link only in the LinkedIn Newsletter/Article native surface, which carries no penalty; or standalone-insight repost of the Substack core point). **Do not change the CLAUDE.md rule from this loop.**

## Substack
**Role:** the crawlable AI-citation surface + deep audience. **Source of truth = `substack_aeo_rules.md`.**
- Every article: question-shaped SEO title (separate from display), one quotable answer sentence up top, outbound link to the rxbs.org canonical.
- Notes: 3/day, origin-story/personal weighted (the only format generating reciprocal engagement). **2026 consensus: Notes is the #1 organic discovery engine** (surfaced to non-subscribers); growth writers attribute the majority of their growth to Notes. Vendor "5-7/day minimum" advice exists but PBS holds at 3/day — measured cadence + brand-fit (volume past 3 reads as influencer-churn and trips the Humanize Check). _(DIRECTIONAL — [thrivewithcarrie](https://thrivewithcarrie.substack.com/p/substack-notes-strategy-2026), [Sprout Social](https://sproutsocial.com/insights/substack-seo/))_
- **SEO is real on Substack** (Google crawls it; Substack domain authority lets hero posts rank). Reinforces the question-shaped SEO title + answer-first lead in `substack_aeo_rules.md`. Keep evergreen "hero" pieces (Contract Language Library, glossary twin) as the rankers. _(CONFIRMED — [Sprout Social](https://sproutsocial.com/insights/substack-seo/))_
- **Biggest untapped growth lever: Recommendations** (get recommended by Potter + aligned newsletters) — see `entity_authority_offsite_plan.md`.

## Facebook (Reels + Page) — _newer surface for PBS_
**Role:** secondary distribution; Reels can reach non-followers. Lowest priority of the six for a B2B plan-sponsor audience (skews consumer), so use as a **repost surface for the strongest video clips**, not a bespoke channel.
- **Reels reach non-followers first** (pushed to new audiences to test performance before existing followers), so a strong clip can travel without paid spend — Reels are now one of the strongest organic-reach drivers on Facebook. _(CONFIRMED — [Fanpage Karma](https://www.fanpagekarma.com/insights/the-facebook-algorithm-2026-explained-what-marketers-need-to-know/), [QuickFrame](https://quickframe.com/blog/facebook-reels-guide-for-marketers))_
- **Completion rate is king:** a 15s Reel most people finish beats a 60s Reel they swipe; viewers decide in <3s. Educational/inspirational videos up to ~90s outperform promotional ones. _(directional)_
- **Saves and DM-shares are the most powerful engagement signals** (above likes). Long comments + shares > reactions. _(CONFIRMED — [Fanpage Karma](https://www.fanpagekarma.com/insights/the-facebook-algorithm-2026-explained-what-marketers-need-to-know/), [SocialBee](https://socialbee.com/blog/facebook-algorithm/))_
- **Consistency compounds:** accounts posting consistently saw ~5x more engagement per post; consistent format/topic helps the algorithm predict engagement. _(directional)_
- Meta **deprioritizes** low-value reposts, reaction clips without commentary, and stitches; overly promotional content is penalized (favor ~80/20 value-to-promo). _(directional)_
- **PBS fit:** repurpose the same captioned 9:16 clip used for Shorts/TikTok; do not build Facebook-first. Revisit priority only if analytics show plan-sponsor reach.
- Sources: [Fanpage Karma](https://www.fanpagekarma.com/insights/the-facebook-algorithm-2026-explained-what-marketers-need-to-know/), [QuickFrame](https://quickframe.com/blog/facebook-reels-guide-for-marketers), [SocialBee](https://socialbee.com/blog/facebook-algorithm/), [Postoria](https://postoria.io/blog/facebook-algorithms).

## TikTok — _planned video surface; treat as reach/discovery_
**Role (planned):** top-of-funnel reach + discovery for the podcast-clip pipeline. B2B rarely closes on TikTok, but it builds high-intent interest **when the next step is frictionless** (bio link → newsletter/toolkit). Skew to POVs, frameworks, opinionated takes the audience can reference.
**Current best practices (2026):**
- **Hook in the first 1-3 seconds; retention is the dominant signal** — first-3-second retention now matters more than follower count. Vendor sources cite a **~70%+ completion target to go viral in 2026**. _(directional, [Hootsuite](https://blog.hootsuite.com/tiktok-algorithm/), [Darkroom](https://www.darkroomagency.com/observatory/how-tiktok%E2%80%99s-algorithm-works-in-2026-and-15-tactics-to-go-viral))_
- **Length (2026 refinement): ~21-34s gets the most engagement;** length only matters via its *relationship to retention* (a 12s clip watched to the end + replayed beats a 3-min clip people drop at 30s). Earlier "60-180s outperforms 15s" guidance is superseded by the retention-relative framing. **Start PBS clips at ~21-34s.** _(directional — [Darkroom](https://www.darkroomagency.com/observatory/how-tiktok%E2%80%99s-algorithm-works-in-2026-and-15-tactics-to-go-viral), [Mickyweis](https://www.mickyweis.com/en/tiktok-algorithm-2026/))_
- **Replays/rewatches are an explicit 2026 ranking lever.** A loopable or "watch-again" payload (a decoder frame worth a second look) helps. _(directional)_
- **Audio:** trending sounds give a boost, but **original audio is favored**; if using a trend, use it within ~24h of it trending. _(directional)_
- **Hashtags (2026 refinement): 1-2 trending + 1-2 niche + 1 branded; hashtags are now LESS critical** — keyword-rich captions, on-screen text, and voiceover carry equal-or-greater discovery weight (TikTok reads the video itself). Treat TikTok as a search engine. _(directional — [Accio](https://www.accio.com/business/tiktok-hashtag-best-practices-2026-trend), [Skedsocial](https://skedsocial.com/blog/how-to-use-hashtags-on-tiktok-in-2026-maximize-your-tiktok-reach-and-engagement))_
- **Niche-relevance > broad reach:** the 2026 algorithm favors community-aligned, native, original content over reposts. Adding a new perspective beats reposting. _(CONFIRMED across sources)_
- **Legit "hacks" to test (ToS-safe), each pending brand-fit:** photo/carousel posts (strong reach lane, good for a "5 things" decoder); reply-to-comment videos (mine the comments for follow-on posts); TikTok Series (gated deeper content); posting in the first 24h of a trend. **Avoid:** engagement-pod/follow-for-follow schemes, reposting others' clips, watermarked cross-posts (TikTok suppresses visible competitor watermarks).
- **PBS fit:** the named-adversary / "guess which…" decoder shapes that work on X translate well; lead with the confrontation, keep it native (not a polished ad), captions burned in. Bio link → newsletter or a toolkit landing page so the next step is frictionless.
- Sources: [Buffer](https://buffer.com/resources/tiktok-algorithm/), [Sprout Social](https://sproutsocial.com/insights/tiktok-algorithm/), [Hootsuite](https://blog.hootsuite.com/tiktok-algorithm/), [Socialync](https://www.socialync.io/blog/tiktok-algorithm-2026-what-works-now), [Socialinsider B2B](https://www.socialinsider.io/blog/tiktok-for-b2b/), [editorialge B2B](https://editorialge.com/tiktok-for-b2b/).

## YouTube Shorts — _planned video surface; you don't know it yet, so start here_
**Role (planned):** evergreen, searchable discovery (Shorts surface + YouTube search) → channel subscribers. Longer shelf life than TikTok (Shorts keep getting served for weeks/months), which suits evergreen PBM explainers.
**Current best practices (2026):**
- **Watch-through rate is THE metric.** The algorithm weighs watch time, swipe-aways, and how fast people leave. ~3 seconds to hook. _(CONFIRMED across sources)_
- **Length (2026 refinement): 15-30s consistently hits the highest retention (often >80%);** the algorithm's "good retention" threshold is now **~70%+ watch-through**, and a 30s Short at 85% retention outranks a 60s Short at 50% every time. Shorts can run to 3 min but completion rules. **Start PBS Shorts at ~15-30s** (tightened from the prior 20-40s). _(CONFIRMED across [Shortimize](https://www.shortimize.com/blog/youtube-shorts-retention-rate), [OpusClip](https://www.opus.pro/blog/ideal-youtube-shorts-length-format-retention), [Piktochart](https://piktochart.com/blog/how-long-youtube-shorts/))_
- **Watch-through is channel-wide:** if average view duration sits below ~70% on most uploads, the whole channel's distribution ceiling drops — so a weak Short hurts the next one. _(directional — [Shortimize](https://www.shortimize.com/blog/youtube-shorts-retention-rate))_
- **First 48 hours get a temporary boost** → consistent posting cadence matters; regular posting trains the algorithm to test you with new viewers. _(directional)_
- **Interaction now ranks Shorts** more: comments, shares, **remixes**, replays. End with a question to drive comments. _(directional)_
- **Seamless loop** (last frame flows into the first) lifts replays/watch-through. **No competitor watermarks** (TikTok logo etc. suppresses Shorts). Vertical 9:16 1080×1920, burned-in captions (most watch muted). _(CONFIRMED)_
- **End-screen / CTA → subscribe** (Shorts → channel growth is the model); put the conversion link in the channel/bio and pinned comment, not the Short body.
- **Discovery differs from TikTok:** Shorts leans more on **title + search relevance** (it's YouTube), so write a real keyword title; TikTok leans on the FYP + audio. Same clip, different title/tag treatment.
- **PBS fit:** evergreen decoder clips ("What is spread pricing", "5 admin fees decoded") fit Shorts' long shelf life and search behavior perfectly; pair each Short with a keyword title matching a buyer question (the AEO logic, in video form) and an end-card to the newsletter.
- Sources: [Social Champ](https://www.socialchamp.com/blog/youtube-shorts-algorithm/), [Metricool](https://metricool.com/youtube-shorts-algorithm/), [vidIQ](https://vidiq.com/blog/post/youtube-shorts-algorithm/), [Riverside](https://riverside.com/blog/youtube-shorts-algorithm), [Miraflow Jan 2026 update](https://miraflow.ai/blog/youtube-shorts-algorithm-update-january-2026).

## Instagram (Reels + feed) — _planned video surface_
**Role (planned):** part of the cross-post video trio (with TikTok + Shorts). Reels reach non-followers; B2B works when treated as a distribution engine, not a brand film. Lower buyer-density than LinkedIn, so a repost/discovery surface for the strongest clips, not bespoke.
- **Top three 2026 ranking signals: watch time, sends-per-reach (DM shares), likes-per-reach** — in that order. Viewers decide in the first ~1.5-2s, so hook instantly. _(directional — [Hootsuite](https://blog.hootsuite.com/instagram-algorithm/), [creatorflow](https://creatorflow.so/blog/instagram-algorithm-2026/))_
- **Length:** in-app Reels cap at 90s (the platform technically allows up to 20 min, but discovery pushes shorter). **7-15s for reach/new-audience** (Reels with **75%+ average watch time** get pushed to non-followers far more); 15-60s for tutorial depth. _(directional — [Hootsuite](https://blog.hootsuite.com/instagram-algorithm/), [SocialBu](https://socialbu.com/blog/instagram-reels-length))_
- **Sends/DMs and Saves are the highest-leverage signals** for reaching new audiences; design the clip to be shared/saved (a quotable decoder frame). _(directional)_
- **Originality is now strongly weighted: original content gets ~40-60% more distribution than reposts, and posting 10+ reposts in 30 days can exclude an account from recommendations entirely.** Cross-posting the same captioned clip is fine; do not lean on reposting others' content. _(directional — [Hootsuite](https://blog.hootsuite.com/instagram-algorithm/))_
- Original content + (optional) trending audio; vertical 9:16; burned-in captions; keyword-rich caption (IG is also a search surface). _(directional)_
- **PBS fit:** repurpose the same captioned 9:16 clip; link in bio (IG suppresses in-caption links); the "5 things / decoder" frames are save-bait. Sources: [Hootsuite](https://blog.hootsuite.com/instagram-reels/), [Gainsty](https://www.gainsty.com/blog/instagram-reels-in-2026), [creatorflow](https://creatorflow.so/blog/instagram-algorithm-2026/).

## Threads — _newer text surface_
**Role:** secondary text/conversation surface (Meta's X competitor); reply-driven reach, growing B2B presence. Low effort if cross-posted from the X/LinkedIn text shapes.
- Favors **conversation and replies / quality discussion** over broadcast; meaningful replies prioritized; fresh content gets priority distribution; time-spent-on-post signals interest. _(CONFIRMED direction — [Planable](https://planable.io/blog/how-to-grow-on-threads/), [Metricool](https://metricool.com/threads-algorithm/))_
- **Link penalty largely reversed in 2026** — Threads now supports link sharing without the earlier suppression many creators saw. So a body link is less risky here than on X/LinkedIn, but keep it conversational, not promo. _(directional — [QuickFrame](https://quickframe.mountain.com/blog/how-does-the-threads-algorithm-work/), [posteverywhere](https://posteverywhere.ai/blog/how-the-threads-algorithm-works))_
- **Cadence:** 2-4 posts/day spaced across peak windows; B2B audiences active weekday mornings + lunch. Questions/polls/topic tags drive the reply-engagement the algorithm rewards. _(directional)_
- **PBS fit:** the named-adversary / decoder one-liners from X translate; thought-leadership + genuine engagement (not pitches) is the B2B mode. Keep it ally-positive. Treat as low-priority cross-post until data justifies more. _(directional)_

## YouTube (long-form) — _the clip SOURCE + a search surface_
**Role:** home for full podcast appearances and interviews (the SHRM / Honest HR / Potter-podcast material) that the Shorts/clips are cut FROM, and a discovery surface (YouTube is the #2 search engine). The channel is the asset library; Shorts feed subscribers to it.
- **2026 shift: YouTube now ranks primarily on viewer SATISFACTION (repeat viewing, survey responses, session continuation), not raw watch time.** Average view duration, CTR, subscriber-gain rate, and share/save rates are the long-form priorities. _(CONFIRMED — [vidIQ](https://vidiq.com/blog/post/understanding-youtube-algorithm/), [Miraflow](https://miraflow.ai/blog/how-youtube-algorithm-works-2026))_
- **Thumbnail is the single biggest CTR factor;** if CTR is <3% after 48h, test a new title/thumbnail (one change at a time). Human face + clear emotion can lift CTR meaningfully. **Get to the main point within the first ~30s** — that opening window is now a core ranking input. _(directional — [vidIQ](https://vidiq.com/blog/post/understanding-youtube-algorithm/), [repindia](https://www.repindia.com/blog/youtube-algorithm-2026/))_
- Add **chapters** (improve searchability + Google/YouTube search snippets), keyword titles matching buyer questions, and an end-screen → newsletter/site. _(CONFIRMED)_
- Publish the full guesting episodes here (where rights allow) so they're searchable and so Shorts can link back to the full piece. _(directional)_
- **PBS fit:** evergreen "explainer" longforms + the podcast catalog; this is the canonical home the Shorts point to. Pairs with the `clip-podcast` pipeline (clips are cut from these).

## Reddit & Quora — _community + AEO citation surface_
**Role:** disclosed expert answers that build the entity AND get cited by AI engines (Google AI Overviews + ChatGPT lean on Reddit/Quora). **Detailed playbook + guardrails already in `entity_authority_offsite_plan.md`.**
- **2026 AEO weight rising sharply:** Reddit crossed **~5%+ of all ChatGPT citations (Jan 2026)**, and **Google AI Overviews source mostly from Reddit (then YouTube, then Quora)**. **May 2026: Google rolled out AI Overviews "Community Perspectives," pulling quotes directly from Reddit/forum threads into results**, so a well-written, on-topic Reddit answer can land verbatim in an AI Overview. _(CONFIRMED — [seroundtable](https://www.seroundtable.com/chatgpt-google-aio-sources-39578.html), [CMSWire](https://www.cmswire.com/digital-marketing/reddits-rise-in-ai-citations-what-marketers-must-know-about-aeo-strategy/), [scion-social](https://scion-social.com/blog/why-smart-companies-are-pivoting-to-reddit-and-quora-and-how-its-affecting-ai-search-rankings/))_
- Answer real questions in **r/benefits, r/HealthInsurance, r/humanresources** and on Quora ("how do I audit my PBM," "is spread pricing legal," "carve in vs out"): give a genuinely useful answer first, disclose the affiliation, link the matching rxbs.org guide/glossary only where it helps. ~1-2/week.
- **Never astroturf** (one helpful disclosed answer beats ten promo ones); Reddit punishes self-promotion hard and **2026 mod detection is reportedly ~40% more accurate than 2025.** Quora norm: ~90% value / 10% subtle promo over months. _(CONFIRMED — community norm + [aileads](https://www.aileads.now/blog/an-honest-look-at-reddit-for-2026), [reddireach](https://www.reddireach.com/blog/reddit-and-quora-distribution-system-for-ai-search-in-2026))_
- **PBS fit:** high. These are buyer-intent questions and an AEO win at once; the Community-Perspectives change *raises* the payoff. The constraint is authenticity + cadence, not reach mechanics. Write each answer as a clean, self-contained, quotable block (it may be lifted verbatim into an AI Overview).

## Not prioritized (for a B2B plan-sponsor audience)

## Cross-platform clip rule (the video pipeline)
One captioned 9:16 clip can feed **LinkedIn (native, Wed slot) → YouTube Shorts (keyword title, end-card) → TikTok (native hook, bio link) → Instagram Reels → Facebook Reels (repost)**, cut from the full episode on **YouTube (long-form)**. Build it once via `/clip-podcast`, then tailor per-platform per the sections above (LinkedIn first per the LinkedIn-first rule). The clip pipeline lives in `social_clips/` + the `clip-podcast` skill.
